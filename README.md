# rembirth

A command-line tool to remember and track birthdays.

## Installation

```bash
cargo build --release
cp target/release/rembirth ~/.local/bin/rembirth
```

## List of Commands

- `name` - Person's name (string)
- `month` - Month of birth (1-12)
- `day` - Day of birth (1-31)
- `year` - Year of birth (e.g., 1990)
